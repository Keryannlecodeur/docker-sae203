🎮 Conception Client & Serveur — PuissanceChat

🖥️ ClientPuissance4IHM

La classe ClientPuissance4IHM permet à un joueur de se connecter à un serveur pour jouer à Puissance 4 avec une interface graphique Java Swing. Elle demande l’adresse IP du serveur, puis établit une connexion. Le joueur reçoit son numéro et attend le début de la partie.

L’interface graphique affiche une grille de 6 lignes par 7 colonnes, avec un bouton par colonne pour jouer. Chaque action est transmise au serveur.

Le client reste connecté via une socket : il reçoit les mises à jour de la grille, l'état de la partie, et affiche les tours ou résultats.

🔌 ServeurPuissance4IHM

Le serveur écoute sur le port 8080 et gère deux connexions clients. Chaque joueur se voit attribuer un numéro, et la communication se fait via des sockets.

Une fois les deux clients connectés, il envoie les signaux de début de partie, reçoit les coups joués, met à jour la grille, vérifie les victoires, puis envoie les résultats.

Chaque joueur est géré dans un thread séparé pour permettre des échanges simultanés.
