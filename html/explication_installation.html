⚙️ Installation du Jeu PuissanceChat

🔧 Prérequis

Avant de commencer, assurez-vous que les outils suivants sont installés sur votre machine :

  • Git — Pour cloner le dépôt depuis GitHub
  • Docker — Pour créer et exécuter des conteneurs

🖥️ Installation — Côté Serveur

  1. Cloner le dépôt :
    git clone git@github.com:Keryannlecodeur/docker-sae203.git
  2. Accéder au dossier :
    cd docker-sae203
  3. Convertir le script si nécessaire :
    dos2unix start.sh
  4. Créer l’image Docker :
    docker build -t <nom_conteneur> .
  5. Lancer le serveur :
    docker run -it -p 8080:8080 <nom_conteneur>
    Le serveur sera accessible sur le port 8080.

👤 Installation — Côté Client

  1. Cloner le dépôt :
    git clone git@github.com:Keryannlecodeur/docker-sae203.git
  2. Compiler le client Java :
    javac ClientPuissance4IHM.java
  3. Lancer le client :
    java ClientPuissance4IHM
